# arduino-joystick-snake-game
Proje Hakkında
Bu proje, fiziksel bir Arduino Uno joystick modülü ile kontrol edilen, C# Windows Forms ortamında geliştirilmiş interaktif bir masaüstü yılan oyunudur. Donanım ve yazılım arasındaki anlık veri aktarımı, seri haberleşme (Serial Communication) protokolü üzerinden sağlanmıştır.

Kullanılan Teknolojiler
Donanım: Arduino Uno, Joystick Modülü

Yazılım: C# (Windows Forms), Arduino IDE (Embedded C)

Özellikler
Joystick ile gerçek zamanlı oyun kontrolü.
Seri haberleşme üzerinden donanım-yazılım entegrasyonu.
Dinamik skor takibi.
